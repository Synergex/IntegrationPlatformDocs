Installing Storage Servers

Send comments on this topic

← Previous Next →

The Streaming Integration Platform relies on taking periodic snapshots of your applications data and those snapshots are usually stored on some network-based storage server. The preferred solution is to use some form of S3 compatible storage, although other solutions such as NFS servers are supported in some scenarios.

 

Synergex has tested the the following S3 compatible storage server solutions, both of which are free, or at least have free options.
