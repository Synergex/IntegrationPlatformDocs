Windows 11

 

•Professional or Enterprise.

•The optional Virtual Machine Platform component is enabled.

•The optional Windows Subsystem for Linux component is enabled and installed.

•Windows Terminal (should be present by default in Windows 11)

•User account must have local administrator rights.

 

 

Visual Studio 2022

 

•Any edition that you are licensed for.

 

•Can be installed using WinGet:

 

winget install --id Microsoft.VisualStudio.2022.<edition> -e

 

Where <edition> is one of: Community, Professional or Enterprise.

 

•Install at least the following workloads and components installed:

•.NET Desktop Development

•ASP.NET and Web Development

•Desktop Development with C++

 

•If necessary you can also install Git for Windows by selecting it from the "Individual Components" tab.

 

 

SQL Server 2022

 

•Express or Developer with the following components:

 

•Can be installed using WinGet:

 

winget install --id Microsoft.SQLServer.Developer -e

 

•During installation:

•Configure the database as the default instance (MSSQLSERVER) not a named instance.

•Select the optional PolyBase Query Service for External Data component.

•Select mixed mode authentication (Windows and SQL Server logins) and for the highest compatibility with the default development environment configuration, set the "sa" account password to "manager".

•Make sure your Windows account is added as a SQL Server administrative user.

 

•After installation:

•Use SQL Server Configuration Manager to enable the Named Pipes and TCP/IP network protocols.

•Restart the MSSQLSERVER service.

•Use Windows Defender Firewall with Advanced Security to open TCP/IP port 1433 for inbound access.

 

 

Synergy/DE

 

•Download from: https://resources.synergex.com

•Install the x86 (101), x64 (104) and SDI products.

•You must run a local license server.

•We recommend that you have at least 10 available runtime (RUN12) licenses.

 

 

Git for Windows

 

•Git for Windows can be installed as an optional component when installing Visual Studio.

 

•Or it can be installed using WinGet:

 

winget install --id Git.Git -e

 

•Or download from: https://git-scm.com/downloads/win and installed manually.

 

 

WiX Toolset V3 and Visual Studio Extension (Votive)

 

 

•Download Wix Toolset 3.14.1 from https://github.com/wixtoolset/wix3/releases/tag/wix3141rtm and install manually.

 

•Download the Visual Studio Extension from https://marketplace.visualstudio.com/items?itemName=WixToolset.WixToolsetVisualStudio2022Extension and install manually.

 

 

7-Zip and 7-zip ZS

 

•7zip can be installed using WinGet:

 

winget install 7zip.7zip -e

 

•Or download from: https://www.7-zip.org/download.html and installed manually.

 

ISAM snapshot files are  counted files compressed into LZ4 archives and stored in a TAR file. If you want to be able to extract the LZ4 files on windows then a special different version of 7zip is required. It can be installed side-by-side with the regular version.

 

7zip ZS can be installed using WinGet:

 

winget install mcmilk.7zip-zstd -e

 

•Or downloaded from: https://www.majorgeeks.com/files/details/7_zip_zs.html and installed manually.

 

 

LLVM

 

LLVM (Low Level Virtual Machine) is a compiler infrastructure project that provides a set of modular, reusable compiler and tool chain technologies. It’s widely used in programming language development, code analysis, and optimization.

 

•Can be installed using WinGet:

 

winget install LLVM.LLVM -e

 

 

CMake

 

CMake (cross-platform make) is a cross-platform build system generator.

 

•Can be installed using WinGet:

 

winget install Kitware.CMake -e

 

 

Rust

 

Rust is a modern systems programming language that focuses on performance, reliability, and safety.

 

•Can be installed using WinGet:

 

winget install Rustlang.Rustup -e
