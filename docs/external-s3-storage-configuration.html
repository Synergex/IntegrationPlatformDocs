Configuration /

External S3 Storage Configuration

Send comments on this topic

← Previous Next →

If snapshot data will be stored in an external S3 storage provider then you must provide information about that environment using the S3StorageExternal property in MasterConfig.json.

 

 

Configuration Properties

 

Property

Type

Description

Region

 

String

 

Server

 

String

 

Port

 

Integer

 

Path

 

String

 

ConsolePort

 

Integer

 

UseSsl

 

Boolean

 

Bucket

 

String

 

Id

 

String

 

Key

 

String

 

 

 

Example Configuration

 

{

  "S3StorageExternal": {

    "Region": "us-west-2",

    "Server": "localhost",

    "Port": 9000,

    "Path": "s3",

    "ConsolePort": 9001,

    "UseSsl": true,

    "Bucket": "development",

    "Id": "admin",

    "Key": "p@ssw0rd"

  }

}
