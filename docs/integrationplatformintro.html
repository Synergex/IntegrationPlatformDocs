Introduction

 

For many decades Synergy DBL had been used to build business software applications, and many of these applications continue to be used to operate businesses of all sizes, and across many business sectors.

 

Depending on the hosting platform, the majority of DBL applications store their information in a file-based database, provided either by Record Management Services (RMS) on OpenVMS systems, or by the Synergy Data Management System (SDMS) on other platforms such as Windows, Linux and UNIX. Storing data primarily in "Indexed Sequential Access Method" (ISAM) data files typically results in very high performance for the original applications owning the data, but in many cases is a barrier to integration with other applications, most of which are not able to consume data stored in this way.

 

The Integration Platform is a software environment, developed by Synergex, that facilitates the integration of Synergy DBL-based applications with other application environments, primarily by providing access to the original applications data by replicating it, in near to real time, to some other data storage platform that is more widely accessible.
