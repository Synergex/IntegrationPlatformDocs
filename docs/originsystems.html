Origin Systems

Send comments on this topic

← Previous Next →

An origin system is a system that hosts a software application written in DBL, and where that application stores its data in a file-based database using either the Synergy Data Management System (SDMS) on Windows or Linux, or Record Management Services (RMS) under OpenVMS IA64.

 

The Streaming Integration Platform is not supported on origin systems running on traditional UNIX or OpenVMS Alpha systems because of dependencies on products or libraries that are not available on those platforms.

 

To be compatible with the Streaming Integration Platform the original DBL application must be built using DBL version 12.3.1 or higher.
