HelpSmith

 

HelpSmith is required if you need to work on the documentation. We are currently using version 10.3.

 

HelpSmith can be downloaded from https://www.helpsmith.com/download.php but does require a license to use.

 

 

PuTTY

 

Although you can work with SSH connections to OpenVMS from the Windows command line or Windows Terminal, it's generally better to use an actual terminal emulator, and PuTTY is the one that many people use.

 

•PuTTY can be installed using WinGet:

 

winget install --id PuTTY.PuTTY -e

 

•Or download from https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html and install manually.

 

 

PuTTY Launcher

 

PuTTYLauncher is an app developed by Steve Ives. It helps you associate defined PuTTY sessions with login credentials, and lets you easily connect to and log in to remote sites. It can be configured to run in the windows system tray, making it easy to log into remote systems with just a couple of clicks.

 

•PuTTY Launcher can be downloaded from https://github.com/SteveIves/PuTTYLauncher/releases.

 

 

SQL Server Management Studio

 

You can work with SQL Server directly from Visual Studio by adding a "data connection" in the Server Explorer window, but many prefer to work in a dedicated tool like SSMS, which is more capable.

 

•SSMS can be installed using WinGet:

 

winget install --id=Microsoft.SQLServerManagementStudio -e

 

•Or download it from https://aka.ms/ssms/21/release/vs_SSMS.exe and install manually.

 

 

TortioseGit

 

If you like to perform Git operations from Windows File Explorer then install TortoiseGIT

 

•TortoiseGit can be installed using WinGet:

 

winget install --id TortoiseGit.TortoiseGit -e

 

•Or download it from https://tortoisegit.org/download and install manually.

 

 

Visual Studio Code

 

It's always useful to have a lightweight and capable alternative to Visual Studio around, and many prefer Visual Studio Code.

 

•VS Code can be installed using WinGet:

 

winget install -e --id Microsoft.VisualStudioCode

 

•Or download it from https://code.visualstudio.com/download and install manually.
