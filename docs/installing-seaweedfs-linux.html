Installing & Configuring SeaweedFS on Linux

Send comments on this topic

← Previous Next →

Download the Software

 

Download the latest linux build from https://github.com/seaweedfs/releases

 

(At the time of writing it was 3.85 and the download was linux_amd64.tar.gz

 

Then extract it:

 

$ gunzip linux_amd64.tar.g

$ tar xvf linux_amd64.tar

 

And the file and set ownership:

 

$ sudo mv weed /usr/bin

$ sudo chmod 755 /usr/bin/weed

$ sudo chown root:root /usr/bin/weed

 

Create a user account

 

$ sudo useradd --system --no-create-home --shell /sbin/nologin seaweedfs

 

Create Directories

 

$ sudo mkdir -p /var/lib/seaweedfs/master /var/lib/seaweedfs/volume

$ sudo chown -R seaweedfs:seaweedfs /var/lib/seaweedfs

 

$ sudo mkdir -p /var/lib/seaweedfs/filer

$ sudo chown seaweedfs:seaweedfs /var/lib/seaweedfs/filer

 

Create a systemd service script for the master server

 

Use your favorite text editor to create a new systemd service definition file:

 

/etc/systemd/system/seaweed-master.service

 

And add the following content:

 

[Unit]

Description=SeaweedFS Master Server

After=network.target

 

[Service]

Type=simple

ExecStart=/usr/bin/weed master -mdir=/var/lib/seaweedfs/master -port=9333

Restart=on-failure

User=seaweedfs

Group=seaweedfs

 

[Install]

WantedBy=multi-user.target

 

Create a systemd service script for the volume server

 

Use your favorite text editor to create a new systemd service definition file:

 

$ sudo vi /etc/systemd/system/seaweed-volume.service

 

And add the following content:

 

[Unit]

Description=SeaweedFS Volume Server

After=seaweedfs-master.service

 

[Service]

Type=simple

ExecStart=/usr/bin/weed volume -dir=/var/lib/seaweedfs/volume -max=5 -mserver=10.1.10.21:9333 -port=8080

Restart=on-failure

User=seaweedfs

Group=seaweedfs

 

[Install]

WantedBy=multi-user.target

 

Create a systemd service script for the filer (web UI) server

 

Use your favorite text editor to create a new systemd service definition file:

 

$ sudo vi /etc/systemd/system/seaweed-filer.service

 

And add the following content:

 

[Unit]

Description=SeaweedFS Filer (Web UI)

After=seaweedfs-master.service

 

[Service]

Type=simple

WorkingDirectory=/var/lib/seaweedfs/filer

ExecStart=/usr/bin/weed filer -master=10.1.10.21:9333 -port=8888

Restart=on-failure

User=seaweedfs

Group=seaweedfs

 

[Install]

WantedBy=multi-user.target

 

Create a systemd service script for the S3 gateway server

 

Use your favorite text editor to create a new systemd service definition file:

 

$ sudo vi /etc/systemd/system/seaweed-s3.service

 

And add the following content:

 

[Unit]

Description=SeaweedFS S3 Gateway

After=seaweedfs-filer.service

 

[Service]

Type=simple

ExecStart=/usr/bin/weed s3 -filer=10.1.10.21:8888 -port=8333

Restart=on-failure

 

[Install]

WantedBy=multi-user.target

 

Register and Start the Services

 

$ sudo systemctl daemon-reexec

$ sudo systemctl daemon-reload

$ sudo systemctl enable --now seaweedfs-master seaweedfs-volume seaweedfs-filer seaweedfs-s3
