Source Files

•IOHooksGlobal.dbl (subroutine) - CHECKED FOR DEPENDENCIES CONVERT TO REGULAR SUBROUTINE

•FileInformation.dbl (class) - CHECKED FOR DEPENDENCIES

•ParseFilesConfiguration.dbl (function) - CHECKED FOR DEPENDENCIES

•IOHooksKafka.dbl (class) - CHECKED FOR DEPENDENCIES MUST SHIP AS CODE SO CAN BE EXTENDED

•TimeStamp.dbl (class) - CHECKED FOR DEPENDENCIES

•FileTools.dbl (class) - CHECKED FOR DEPENDENCIES

•ParseJsonFile.dbl () - CHECKED FOR DEPENDENCIES

•KafkaAPI_WL.dbl (functions) - CHECKED FOR DEPENDENCIES

Logical Names

 

Logical Name

Required or Optional

Description

REPLICATOR_LOG

 

 

REPLICATOR_HOOKS_CONSOLE_DEBUG

Optional

Set to YES enables console logging in the I/O hooks.

REPLICATOR_HOOKS_DEBUG_LOG

Optional

Set to YES enables logging in the I/O hooks.

REPLICATOR_HOOKS_LOGLEVEL

Optional

If logging is enabled, defines the level of logging.

REPLICATOR_HOOKS_ENABLED

Required

Set to YES to enable I/O hooks and replication.

REPLICATOR_ENVIRONMENT

Required

Environment (profile) name

REPLICATOR_KAFKA_MAX_FAIL

Optional

Sets the maximum number of Kafka failures before giving up. Default is 1.

REPLICATOR_ZMQ_BROKERS

Required, OpenVMS only

Defines the ZeroMQ brokers used to communicate with the Message Broker.

REPLICATOR_CONFIG

Required

Points to the directory containing the files configuration file for the profile.

REPLICATOR_INCLUDE

Required (for build)

Defines the location of IOHooksKafka.def
